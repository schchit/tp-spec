## Time Protocol (TP) Specifications

The Time Protocol (TP) is a four-layer framework for rewriting time in AI systems.

## The Four Layers

| Layer | Name | Description | Status |
|-------|------|-------------|--------|
| TP/0 | Time Definition | Core concepts and structure | [IETF Draft](https://datatracker.ietf.org/doc/draft-wang-tp-definition-00/) |
| TP/1 | Time Perception | Modulating time density (slow/fast) | Coming soon |
| TP/2 | Time Direction | Directing time flow (future → present) | Coming soon |
| TP/3 | Time Copy | Copying time across agents with accountability | [Implemented by HJS](https://github.com/hjs-spec/spec) |
| TP/4 | Time Emergence | New dimensions emerging from existing ones | Coming soon |
